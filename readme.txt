Lefty Blogs blogroller for WordPress Widgets

Tags: wordpress, leftyblogs, widgets, blog roll, javascript

*What it does*

This plugin allows you to post a LeftyBlogs.com blogroll on your site by bypassing the security feature of Widgets that prevents you from inserting JavaScript code.

*Installation*

1. Extract all files and put the leftyblogs.php file in the /plugins directory
2. Go to your Plugins control panel and activate the plugin
3. Go to Presentation > Widgets to configure the plugin before use.
